Welcome to Salesforce API interface. Use this tool to record useful queries before integrating them into the code.


To get started,
  • First, create a new Salesforce app and configure it to use SAML. Don't forget to add your Salesforce user account profile (or create a new profile) to the app that has sufficient access to account and data.
  • Second, configure this utility with your Salesforce app.
  • Then, authenticate to Salesforce API before being able to execute any queries.
  • Finally, navigate to queries to execute some SOQL searches. Use this reference document to get familiar with SOQL format and examples
    https://developer.salesforce.com/docs/atlas.en-us.api_rest.meta/api_rest/dome_query.htm
